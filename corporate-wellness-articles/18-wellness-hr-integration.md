## Introduction

Wellness programs shouldn't exist in silos. Integration with HR systems creates seamless employee experiences, accurate data, and efficient administration. This guide covers key integration points and implementation considerations.

---

## Key Integration Points

### HRIS Integration

| Data | Direction | Purpose |
|------|-----------|---------|
| Employee demographics | HRIS → Wellness | Auto-enrollment |
| Employment status | HRIS → Wellness | Eligibility updates |
| Department/location | HRIS → Wellness | Segmentation |
| Wellness achievements | Wellness → HRIS | Record keeping |

### Benefits Integration

| Integration | Purpose |
|-------------|---------|
| Eligibility sync | Benefits-linked enrollment |
| Premium discounts | Automatic rate adjustment |
| HSA contributions | Wellness-earned funds |
| Claims data | Population health context |

### SSO (Single Sign-On)

| Benefit | Impact |
|---------|--------|
| One login | No separate credentials |
| Security | Central authentication |
| Adoption | Reduced friction |
| IT management | Unified access control |
