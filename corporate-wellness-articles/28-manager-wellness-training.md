## Introduction

When biometric data indicates an employee is struggling, who responds first? Managers are frontline wellness responders—but most lack training. They need skills to recognize signals, have supportive conversations, and connect employees with resources.

---

## Why Managers Matter

### The Frontline Role

| Responsibility | Impact |
|----------------|--------|
| Daily interaction | First to notice changes |
| Relationship trust | Comfortable conversations |
| Workload control | Can adjust demands |
| Culture creation | Sets team norms |

### Common Gaps

| Gap | Consequence |
|-----|-------------|
| Untrained | Miss warning signs |
| Uncomfortable | Avoid conversations |
| Uninformed | Don't know resources |
| Unsupported | Burnout themselves |

## Training Components

### 1. Recognition Skills

| What to Notice | Possible Indication |
|----------------|---------------------|
| Performance changes | Quality decline |
| Attendance patterns | Increased absences |
| Engagement drop | Withdrawal |
| Communication changes | Tone, responsiveness |

### 2. Conversation Skills

| Skill | Application |
|-------|-------------|
| Non-judgmental inquiry | "How are things going?" |
| Active listening | Hear without fixing |
| Confidentiality | What's private, what's not |
| Avoiding assumptions | Ask, don't guess |

### 3. Resource Knowledge

| Resource | When to Reference |
|----------|-------------------|
| EAP | Mental health support |
| HR | Policy questions |
| Wellness programs | General support |
| Medical leave | When needed |

### 4. Boundary Setting

| Boundary | Reason |
|----------|--------|
| Not a therapist | Professional limits |
| Not an investigator | Trust, not interrogation |
| Not a fixer | Enable, don't solve |
| Self-care | Manager wellness matters |

## Responding to Wellness Alerts

### When System Generates Alert

| Step | Action |
|------|--------|
| 1 | Review aggregate context |
| 2 | Consider what you observe |
| 3 | Plan supportive conversation |
| 4 | Document appropriately |

### Conversation Framework

| Phase | Approach |
|-------|----------|
| Open | Express care, not concern |
| Explore | Open questions |
| Listen | Without judgment |
| Offer | Resources, flexibility |
| Follow up | Continued support |

### What NOT to Do

| Mistake | Why |
|---------|-----|
| Reference specific data | Privacy violation |
| Pressure for disclosure | Damages trust |
| Make assumptions | May be wrong |
| Promise confidentiality you can't keep | Legal/safety limits |
| Ignore the signal | Problems escalate |
