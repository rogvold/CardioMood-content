## Introduction

Traditional elderly care is reactive: wait for symptoms, respond to crises, manage problems after they develop. This model is expensive, stressful, and often ineffective. Proactive care—enabled by continuous monitoring—prevents problems before they become crises.

---

## The Reactive Care Model

### How Traditional Care Works

| Stage | Action |
|-------|--------|
| Senior lives independently | Hope nothing happens |
| Something seems wrong | Worry, wait, watch |
| Symptoms appear | Schedule appointment |
| Crisis develops | ER visit |
| Recovery/decline | Reactive adjustment |

### Problems with Reactive Care

| Issue | Consequence |
|-------|-------------|
| Late detection | Problems advanced |
| Missing data | Decisions without information |
| Crisis-driven | High stress, high cost |
| Episodic | No continuity |
| Family burden | Constant worry |

---

## The Proactive Care Model

### How Proactive Care Works

| Stage | Action |
|-------|--------|
| Continuous monitoring | Capture daily data |
| Pattern detection | AI identifies changes |
| Early alert | Family/caregiver notified |
| Proactive intervention | Doctor visit, adjustment |
| Prevention | Crisis avoided |

### Proactive Advantages

| Advantage | Impact |
|-----------|--------|
| Early detection | 2-3 day warning window |
| Data-informed | Objective information |
| Prevention-focused | Avoid crises |
| Continuous | Always monitoring |
| Family supported | Peace of mind |
