## Introduction

Sleep isn't just rest—it's when the brain clears toxins, consolidates memory, and repairs itself. Sleep disruption in seniors often precedes cognitive decline by months or years. Monitoring sleep patterns provides an early window into brain health.

---

## The Sleep-Cognition Connection

### Why Sleep Matters for Brain Health

| Process | During Sleep |
|---------|--------------|
| Glymphatic clearance | Brain clears amyloid and tau |
| Memory consolidation | Learning becomes permanent |
| Neural repair | Brain cells recover |
| Hormone regulation | Growth hormone, cortisol balance |

### Sleep Problems and Dementia

| Finding | Research |
|---------|----------|
| Sleep apnea | 2x dementia risk |
| Short sleep (<6 hours) | Increased cognitive decline |
| Fragmented sleep | Accelerated brain aging |
| REM reduction | Early Alzheimer's marker |

## What Monitoring Reveals

### Sleep Quality Metrics

| Metric | Insight |
|--------|---------|
| Total sleep time | Duration adequacy |
| Sleep efficiency | Time asleep vs. in bed |
| Wake episodes | Sleep fragmentation |
| Deep sleep | Restorative sleep |
| REM sleep | Cognitive processing |
| Breathing patterns | Sleep apnea detection |

### Warning Patterns

| Pattern | Possible Indication |
|---------|---------------------|
| Declining deep sleep | Normal aging or pathology |
| Increasing wake episodes | Fragmentation concern |
| REM changes | Early cognitive marker |
| Breathing disturbances | Sleep apnea |
| Nighttime activity | Wandering, confusion |

## Early Detection

### Sleep as Biomarker

| Stage | Sleep Changes |
|-------|---------------|
| Normal aging | Gradual, predictable |
| Early cognitive changes | Specific pattern shifts |
| Mild cognitive impairment | Pronounced disruption |
| Dementia | Severe dysregulation |

### Detection Timeline

| Detection | Window |
|-----------|--------|
| Pattern changes | Months to years before diagnosis |
| Progressive decline | Trackable over time |
| Intervention opportunity | While still modifiable |
