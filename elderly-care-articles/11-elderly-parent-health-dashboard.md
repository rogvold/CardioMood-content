## Introduction

You want to know your parent is okay—but you don't have hours to analyze health data. The family dashboard delivers peace of mind in 30 seconds: a quick glance tells you everything you need to know, with details available if you want them.

---

## The 30-Second Check

### What You See First

| Element | Information |
|---------|-------------|
| Overall status | Green/yellow/red at a glance |
| Last night's sleep | Quality summary |
| Today's activity | Movement level |
| Vital sign status | Any concerns |
| Alerts | What needs attention |

### Dashboard Design

| Principle | Implementation |
|-----------|----------------|
| Simplicity first | Status before detail |
| Exception-based | Highlight what's different |
| Color-coded | Visual quick read |
| Progressive detail | Drill down if wanted |

## Alert System

### Alert Levels

| Level | Display | Action Needed |
|-------|---------|---------------|
| All Clear | Green check | None |
| Attention | Yellow dot | Monitor |
| Concern | Orange alert | Check in |
| Alert | Red notification | Contact doctor |
| Urgent | Red urgent | Immediate action |

### What Triggers Alerts

| Trigger | Threshold |
|---------|-----------|
| HRV decline | >15% below baseline |
| Sleep disruption | Significant quality drop |
| Activity drop | Well below typical |
| Vital sign change | Sustained difference |
| Combined patterns | Multiple indicators |
