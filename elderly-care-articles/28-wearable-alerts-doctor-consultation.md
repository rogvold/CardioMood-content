## Introduction

Your parent's health monitoring generates an alert. Should you call the doctor? Not every alert needs medical attention, but some definitely do. Knowing the difference prevents both over-reaction and dangerous delay.

---

## Alert Response Framework

### General Principles

| Principle | Guidance |
|-----------|----------|
| Context matters | Consider circumstances |
| Trends over snapshots | Pattern more meaningful |
| Combined signals | Multiple indicators more serious |
| Know baselines | Understand their normal |
| Trust the system | Designed for meaningful alerts |

### When NOT to Call

| Situation | Rationale |
|-----------|-----------|
| Single minor deviation | Normal variation |
| Expected after activity | Appropriate response |
| Returns to normal quickly | Self-correcting |
| Known cause | Travel, illness, etc. |
| Attention-level alert | Designed for watching |

---

## When to Call

### Definite Doctor Call Situations

| Situation | Reason |
|-----------|--------|
| Alert-level notification | System says it's significant |
| Multiple declining metrics | Pattern of concern |
| Sustained abnormality | Not self-correcting |
| Combined with symptoms | Data + feeling unwell |
| Significant deviation | >15% from baseline |

### Urgent Care/ER Situations

| Situation | Action |
|-----------|--------|
| Urgent-level alert | Immediate attention |
| Critical vital signs | Seek emergency care |
| Unable to reach parent | Check in person |
| Parent reports emergency symptoms | Call 911 |
| Multiple critical indicators | Emergency response |

## Metric-Specific Guidelines

### Heart Rate

| Finding | Response |
|---------|----------|
| Slightly elevated | Monitor |
| 10-15 bpm above baseline sustained | Doctor call within day |
| Very elevated + symptoms | Urgent care |
| Signs of irregular rhythm | Doctor call |

### HRV

| Finding | Response |
|---------|----------|
| 5-10% decline | Monitor |
| 10-15% sustained decline | Doctor discussion |
| >20% decline | Doctor call |
| Combined with other metrics | Increased urgency |

### Sleep

| Finding | Response |
|---------|----------|
| One poor night | Normal variation |
| Several poor nights | Check in with parent |
| Week+ of poor sleep | Doctor discussion |
| Severe disruption | Medical evaluation |

### Activity

| Finding | Response |
|---------|----------|
| Lower day | Normal |
| Several days declining | Check on parent |
| Significant sustained drop | Doctor evaluation |
| Unable to do usual activities | Medical assessment |

### Information to Have

| Prepare | Purpose |
