## Introduction

Your parent doesn't need to be tech-savvy to use health monitoring. Modern devices are designed for simplicity—and with the right approach, setup takes just 10 minutes. Here's how to get your non-tech senior started.

## Before You Start

### What You Need

| Item | Purpose |
|------|---------|
| Wearable device | Health tracking |
| Charger | Device charging |
| Smartphone (yours or theirs) | Initial setup |
| WiFi password | Connectivity |
| Account credentials | Family access setup |

### Prepare the Senior

| Conversation | Purpose |
|--------------|---------|
| Explain the purpose | Why this helps |
| Address concerns | Privacy, difficulty |
| Set expectations | What it will do |
| Offer reassurance | Simple, unobtrusive |

## The 10-Minute Setup

### Minutes 1-2: Physical Setup

| Step | Action |
|------|--------|
| Unbox device | Remove packaging |
| Charge if needed | May ship partially charged |
| Size wristband | Adjust for comfort |
| Test fit | Comfortable, not tight |

### Minutes 3-5: App Setup

| Step | Action |
|------|--------|
| Download app | From App Store/Play Store |
| Create account | Or link to existing |
| Pair device | Follow on-screen steps |
| Enable permissions | As prompted |

### Minutes 6-8: Senior's Device

| Step | Action |
|------|--------|
| Put on wearable | Show correct position |
| Show on/off (if applicable) | Usually always on |
| Explain charging | When and how |
| Test comfort | Adjust if needed |

### Minutes 9-10: Family Access

| Step | Action |
|------|--------|
| Add family members | Share access |
| Configure alerts | Who gets notified |
| Test dashboard | Confirm data flowing |
| Show senior their data | If interested |

## Making It Senior-Friendly

### Device Considerations

| Feature | Why It Matters |
|---------|----------------|
| Large display | Easy to see time |
| Simple interface | No complex navigation |
| Long battery | Less frequent charging |
| Water resistant | Can shower with it |
| Comfortable | Will actually wear it |

### Simplifying for Seniors

| Principle | Implementation |
|-----------|----------------|
| Passive operation | No daily actions required |
| Automatic sync | No manual steps |
| Subtle notifications | Not confusing |
| Minimal charging | Extended battery life |
