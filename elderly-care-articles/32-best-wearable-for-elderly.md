## Introduction

Not all wearables work for seniors. Tiny screens, complex interfaces, and frequent charging frustrate elderly users—and frustrated users stop wearing devices. The right wearable for your parent prioritizes simplicity, comfort, and relevant health metrics.

---

## Key Selection Criteria

### Must-Have Features

| Feature | Why |
|---------|-----|
| Long battery life | Less frequent charging (5+ days ideal) |
| Comfortable design | Will actually wear 24/7 |
| Simple interface | Minimal confusion |
| Heart rate monitoring | Core health metric |
| Sleep tracking | Important for elderly health |
| Water resistance | Can shower, no worries |

### Nice-to-Have Features

| Feature | Value |
|---------|-------|
| HRV monitoring | Deeper health insights |
| Activity tracking | Mobility monitoring |
| Fall detection | Safety feature |
| Large display | Easier to see |
| Quick charging | When needed |

## Key Takeaways

1. **Simplicity first:** Less is more for seniors
2. **Battery life matters:** 5+ days ideal
3. **Comfort essential:** Must wear it constantly
4. **Right metrics:** Heart rate, sleep, activity
5. **Easy charging:** Simple routine
6. **Consider limitations:** Vision, dexterity, cognition
7. **Test before committing:** Try it on
8. **Family access:** Easy data sharing
