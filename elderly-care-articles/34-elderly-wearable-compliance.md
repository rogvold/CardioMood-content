## Introduction

A wearable only works if worn. Seniors who find devices uncomfortable, confusing, or inconvenient stop wearing them—and all monitoring value disappears. Optimizing for compliance means prioritizing comfort, simplifying charging, and removing friction.

---

## The Compliance Challenge

### Why Seniors Stop Wearing

| Reason | Frequency |
|--------|-----------|
| Discomfort | 30% |
| Charging hassle | 25% |
| Forgot to put back on | 20% |
| Skin irritation | 10% |
| Didn't see the point | 10% |
| Technical problems | 5% |

### Impact of Non-Compliance

| Consequence | Problem |
|-------------|---------|
| Data gaps | Missing critical information |
| Delayed detection | Lose early warning |
| False confidence | Family thinks monitoring active |
| Wasted investment | Device not providing value |

## Comfort Optimization

### Physical Comfort

| Factor | Optimization |
|--------|--------------|
| Band material | Silicone or fabric, hypoallergenic |
| Fit | Snug but not tight |
| Weight | Lighter is better |
| Size | Appropriate for wrist size |
| Wear position | May need adjustment |

### Addressing Discomfort

| Issue | Solution |
|-------|----------|
| Skin irritation | Change band, give breaks |
| Too tight | Loosen one notch |
| Too loose | Tighten for accuracy |
| Catches on things | Different form factor |
| Sleep interference | Lighter device for night |

### Long-Term Comfort

| Practice | Purpose |
|----------|---------|
| Regular band cleaning | Prevent irritation |
| Periodic skin breaks | If needed |
| Band replacement | When worn |
| Position rotation | Reduce irritation |

## Battery Management

### Why Battery Matters

| Factor | Impact on Compliance |
|--------|---------------------|
| Charging frequency | More charging = more chances to forget |
| Routine disruption | Breaks wearing habit |
| Dead battery | No monitoring during gap |
| Charging complexity | Frustration source |

### Optimizing Battery Life

| Strategy | Effect |
|----------|--------|
| Choose long-battery device | 5+ days ideal |
| Disable unused features | Extend battery |
| Consistent charging time | Build routine |
| Quick charging | Minimize off time |

### Establishing Charging Routine

| Approach | Implementation |
|----------|----------------|
| Same time daily | During shower/bath |
| Same location | Dedicated charging spot |
| Visual reminder | Charger in obvious place |
| Family reminder | Initial support |

## Building Wearing Habit

### First Two Weeks

| Week | Focus |
|------|-------|
| Week 1 | Daily reminders, address issues |
| Week 2 | Reduce reminders, reinforce habit |

### Habit Reinforcement

| Strategy | Purpose |
|----------|---------|
| Morning routine | Part of getting dressed |
| Visual check | Family asks "got your watch?" |
| Positive feedback | Comment on data collected |
| Show value | Share interesting insights |

## Monitoring Compliance

### Signs of Non-Compliance

| Indicator | What to Check |
|-----------|---------------|
| Data gaps | Not wearing |
| No overnight data | Taking off at night |
| Sporadic data | Inconsistent wear |
| Battery not charging | Device forgotten |

### Addressing Issues

| Issue | Response |
|-------|----------|
| Discomfort | Try adjustments, different device |
| Charging forgot | Reminder system |
| Lost interest | Re-emphasize value |
| Skin issues | See dermatologist if needed |

## Tips for Long-Term Success

### Family Support

| Action | Value |
|--------|-------|
| Regular check-ins | Catch issues early |
| Positive reinforcement | Encourage wearing |
| Problem-solving | Address challenges |
| Show appreciation | For participating |

### Senior Engagement

| Strategy | Effect |
|----------|--------|
| Show their data | Creates ownership |
| Celebrate progress | Motivates continued use |
| Involve in choices | Autonomy |
| Listen to feedback | Address real concerns |

---

## Key Takeaways

1. **Comfort is critical:** Discomfort = non-compliance
2. **Battery matters:** Longer = better compliance
3. **Establish routine:** Same time, same place charging
4. **First weeks crucial:** Build habit early
5. **Monitor for gaps:** Catch non-compliance
6. **Address issues quickly:** Before habit breaks
7. **Show value:** Why it matters to them
8. **Family support:** Ongoing encouragement
