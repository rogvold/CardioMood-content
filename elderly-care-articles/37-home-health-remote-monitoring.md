## Introduction

Home health visits are periodic—but patient health is continuous. Remote monitoring extends visibility between visits, enabling earlier detection, better care coordination, and differentiated services. For agencies, it's both clinical enhancement and competitive advantage.

---

## The Home Health Opportunity

### Current Model Limitations

| Limitation | Impact |
|------------|--------|
| Visit-based only | Gaps between assessments |
| Limited visibility | Miss changes between visits |
| Reactive response | Respond to problems, not prevent |
| Documentation burden | In-person only |

### Monitoring Enhancement

| Capability | Value |
|------------|-------|
| Between-visit visibility | Continuous health data |
| Early detection | Catch problems sooner |
| Proactive intervention | Prevent deterioration |
| Remote documentation | Automated data collection |
