## Introduction

"Does Medicare cover this?" is a common question when considering elderly health monitoring. The coverage landscape is evolving—with expanding reimbursement for remote patient monitoring. Understanding what's covered helps families plan and access available benefits.

---

## Medicare Coverage

### Remote Patient Monitoring (RPM)

| Coverage Element | Status |
|------------------|--------|
| CPT codes exist | 99453, 99454, 99457, 99458 |
| Covered services | Setup, monitoring, review |
| Provider requirement | Must be ordered by physician |
| Time thresholds | Minimum monitoring time |

### What's Covered

| Service | Coverage |
|---------|----------|
| Device setup | CPT 99453 |
| Device supply/data transmission | CPT 99454 |
| Treatment management | CPT 99457, 99458 |

### Requirements

| Requirement | Detail |
|-------------|--------|
| Physician order | Must be prescribed |
| FDA-cleared device | Medical-grade required |
| Chronic condition | Ongoing management |
| Time documentation | Minimum thresholds |

### Limitations

| Limitation | Implication |
|------------|-------------|
| Must be chronic condition | Not for general wellness |
| Provider involvement | Not consumer self-monitoring |
| Device requirements | FDA clearance needed |
| Documentation | Specific requirements |
